Investigate the Command: $ python3 -m venv ~/myherozero

1. The python3 command opens a python interactive sesssion of that specific version
2. -m command opens a module?
3. Executing the venv command creates a lightweight "virtual environment" at the target location
4. ~/myherozero indicates the path to that location
5. ~/ is shorthand for the current user's home folder
